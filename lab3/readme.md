# Лабораторная работа №3

## Задача

Реализовать очередь на основе двух стеков.

Необходимо реализовать методы:

* `enqueue(x)` — добавление элемента;
* `dequeue()` — удаление первого элемента;
* `front()` — получение первого элемента без удаления;
* `clear()` — очистка очереди.

## Описание решения

Для работы используются два стека:

* `stack_in` — для добавления элементов;
* `stack_out` — для извлечения элементов.

При необходимости элементы из `stack_in` переносятся в `stack_out`, что позволяет сохранить порядок FIFO (первый пришёл — первый вышел).

## Пример работы

```python
q.enqueue(10)
q.enqueue(20)
q.enqueue(30)

print(q.front())      # 10
print(q.dequeue())    # 10
print(q.dequeue())    # 20

q.clear()
```
